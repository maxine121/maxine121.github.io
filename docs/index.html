Projeto EDOO

Sistema de Reservas de Salas do Cin

Descrição do projeto

Nosso projeto consiste em um sistema CRUD de reservas de salas do Cin. Nele o usuário consegue agendar reservas, listar reservas atuais e, caso for administrador, cancelar reservas e adicionar novas salas.

Criação de reservas de salas

Processo de criação de reservas com base nas salas existentes e suas respectivas disponibilidades.

Persistência de Dados por meio de documentos txt

Os dados são salvos em documentos txts que são carregados aos vetores do sistema no processo de inicialização.

Deleção de reservas e adição de salas pelo administrador

O administrador consegue cancelar reservas e adicionar novas salas ao banco de dados.
